Cafe Management System

A Java Swing-based desktop application for managing cafe orders and billing.

Features
- Menu display
- Order management
- Bill generation
- GUI interface

Technologies Used
- Java
- Swing
- OOP

How to Run

1. Install Java JDK
2. Compile:

```bash
javac CafeManagementSystem.java
```

3. Run:

```bash
java CafeManagementSystem
```
